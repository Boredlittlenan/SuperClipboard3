# Changelog

## v2.0.0 (2026-06-27)

### 新功能

- **主题三合一按钮**：将原来的「跟随系统」开关 + 浅色/深色按钮合并为一个三段式切换按钮（浅色 / 深色 / 自动），操作更直观
- **备忘录视觉重设计**：备忘录模块使用暖色调（amber/金色）独立配色，圆角卡片样式，在所有主题下都清晰可辨
- **自定义备忘录颜色**：设置面板新增颜色选择器，支持 8 种预设色 + 自定义 HEX 输入，独立于主题色
- **备忘录归档**：备忘录支持软删除归档，归档 Tab 内分「剪贴板」和「备忘录」两个子标签，30 天自动清理
- **归档倒计时**：归档条目显示剩余天数（30 天自动过期），黄色徽章提示
- **跟随模式**：快捷键唤起窗口时自动定位到鼠标附近（托盘点击不触发），可在设置中开关
- **点击即粘贴**：快捷键唤起窗口后点击条目，自动隐藏窗口并模拟 Ctrl+V 粘贴到之前的活动窗口
- **备忘录拖拽排序**：使用 Pointer Events 重写拖拽（替代 HTML5 Drag-and-Drop），在 Tauri WebView2 中更稳定
- **智能新建按钮**：备忘录新建按钮支持切换行为——正在编辑空备忘录时再次点击关闭编辑器，有内容时保存并关闭
- **中文应用标题**：中文语言下应用标题显示为「小楠の剪切板」

### 改进

- **分类算法升级**：
  - 新增 Markdown 内容检测（标题、加粗、链接、列表等）
  - 新增 YAML/TOML 配置文件检测
  - 新增正则表达式字面量检测
  - JSON 检测更严格：要求带引号的键名，避免误判
  - 分类阈值从 4 分提高到 5 分，减少误分类
  - 自然语言惩罚降低（-2 分），仅对单行文本生效
- **Stats 补全**：统计接口现在包含邮箱和文件路径类别计数
- **代码重构**：提取共享 TrashIcon 组件、去重 SQLite 行映射逻辑、统一全局快捷键回调
- **设置面板分组**：重新组织为「系统设置」和「功能设置」两个区域，视觉更清晰
- **条目背景固定**：所有条目（剪贴板 + 备忘录）使用固定灰白色 `#f5f5f5`，不随主题变化
- **备忘录文字可选**：备忘录内容区域支持文字选中复制（拖拽手柄和操作按钮仍不可选）
- **归档图标适配**：开启归档功能后，删除按钮自动切换为归档图标（📦/📁）
- **编辑徽章颜色统一**：备忘录「已编辑」徽章改用 memo-contrast 配色，与备忘录模块风格一致

### 修复

- **Hover 残留**：删除条目后，下方条目无需移出再移入即可正常触发 hover（改用 CSS `:hover` 替代 React state）
- **拖拽可靠性**：备忘录拖拽从 HTML5 Drag-and-Drop 改为 Pointer Events，解决 WebView2 中拖拽失灵的问题
- **备忘录列表乱跳**：修复切换条目时列表顺序异常的问题

### 清理

- 移除未使用的翻译键（`memoDrawerTitle`、`daysAgo`、`shortcutHint`）
- 移除未使用的 CSS 动画（`slideInRight`、`fadeIn`）
- 修复 i18n 上下文默认值与 `DEFAULT_LOCALE` 不一致的问题
- 移除前端未使用的 import 和死代码分支

## v1.5.0 (2026-06-23)

### 新功能

- **剪贴板内容编辑**：点击条目上的编辑按钮可修改剪贴板内容，编辑后显示「最后编辑时间」，原内容折叠保留，可通过下拉按钮展开查看
- **原格式预览开关**：设置面板新增「原格式预览」选项，开启后以等宽字体完整显示剪贴板原始内容
- **自动检查更新**：设置面板新增「自动检查更新」开关，开启后每次启动应用自动检查新版本，版本号显示在设置面板右上角

- **备忘录功能**：新增独立备忘录模块，与剪贴板分开存储（SQLite `memos` 表）
  - 支持标题、正文、标签
  - 支持搜索、置顶、删除
  - 每次按键实时保存到数据库，切换条目自动加载最新数据
  - 可在设置中开启/关闭备忘录入口
  - 备忘录标签位于"全部"左侧，使用紫色主题

- **窗口置顶开关**：设置面板新增"窗口置顶"选项，默认开启
  - 切换后立即生效，设置持久化到 SQLite

- **数据库大小显示**：底栏实时显示数据库文件占用空间

### 改进

- **默认快捷键改为 Shift+V**：更简短的唤起快捷键（可在设置中自定义）
- **标签栏滚轮滑动**：Tab 栏支持鼠标滚轮水平滚动，应对标签过多溢出的场景
- **剪贴板分类器重构**：
  - 从单一正则匹配改为评分制，多信号加权判断
  - 新增 CSS 检测：属性声明（`property: value;`）、选择器块（`.class { ... }`）
  - 新增 HTML/XML 标签检测
  - 新增 SQL 关键词检测（SELECT / INSERT / JOIN 等）
  - 新增 Shell 命令检测（`sudo`, `npm`, `docker`, `git` 等）
  - 扩充代码关键词覆盖：Rust、Go、Python、JS/TS、PHP、Java 等
  - 支持单行代码片段识别（不再要求必须多行）
  - 自然语言文本不会被误判为代码

### 修复

- **UTF-8 截断崩溃**：修复中文等多字节字符在生成预览时因字节截断导致 panic（改用 `chars().take(n)`）
- **备忘录切换数据不刷新**：点击其他备忘录条目后，编辑器始终从数据库重新加载最新数据，而非使用内存中的旧值
