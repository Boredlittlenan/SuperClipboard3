# SuperClipboard

[English](README.md)

中文名：超级剪贴板。基于 Rust + Tauri + React + TypeScript 构建的轻量级剪贴板管理器。

## 功能特性

- 智能分类：自动识别剪贴板内容类型（文本、链接、图片、代码、邮箱、文件路径）
- 实时监控：SHA-256 哈希去重，避免重复存储
- SQLite 持久化存储，支持索引查询，搜索快速
- 置顶重要条目，一键复制回剪贴板
- 备忘录功能：独立标签页，支持标题/正文/标签/搜索/置顶/自动保存，默认关闭，可在设置中开启
- 全局快捷键显示/隐藏窗口（默认：Alt+X，可在设置中自定义）
- 窗口置顶开关（默认关闭），开启后窗口始终显示在最前面
- 系统托盘集成，右键菜单支持打开设置和退出应用
- 单实例启动：反复点击快捷图标会唤起已有窗口，不再生成重复托盘图标
- 主题模式默认跟随系统，支持浅色 / 深色 / 自动三段式切换
- 设置面板，支持中英文语言切换；首次启动自动跟随系统语言
- 开机自启动（Windows 注册表）
- 用户偏好设置持久化存储（SQLite）
- 标签栏支持鼠标滚轮水平滑动
- 底栏存储分开显示：备忘录页显示备忘录内容占用，剪贴板页显示剪贴板内容占用
- 备忘录图片粘贴：在备忘录编辑区 Ctrl+V 粘贴图片，预览区直接渲染图片，不显示 base64 文本
- 剪贴板内容编辑：在线编辑，自动保留原内容并可折叠展开对比
- 原格式预览开关：以等宽字体完整显示剪贴板内容，不截断；该开关只影响剪贴板，备忘录默认原格式预览
- 自动检查更新开关：默认开启，可在设置中关闭启动时自动检查更新
- 时间显示优化：超过 24 小时的条目显示具体日期时间（如 "6/24 15:30"）
- 紧凑设置面板：各选项悬浮显示详细说明
- 主题三合一按钮：单行三段式切换（浅色 / 深色 / 自动）
- 备忘录独立视觉风格，初始配色为 `#3f3f3f`，不随主题配色变化
- 自定义备忘录颜色：8 种预设色 + HEX 自定义输入，独立于主题
- 回收站：软删除，回收站 Tab 内分「剪贴板」和「备忘录」两个子标签，30 天自动清理，默认关闭
- 回收站倒计时：条目显示剩余天数，黄色徽章提示
- 窗口位置：软件启动时与托盘右键菜单「设置」使用相同默认位置；快捷键和托盘左键唤起时保持用户拖动后的当前位置
- 点击即粘贴：快捷键唤起后点击条目，自动隐藏窗口并模拟 Ctrl+V 粘贴到活动窗口
- 备忘录拖拽排序：Pointer Events 实现，在 Tauri WebView2 中稳定可靠
- 应用改名迁移：从 `SuperClipboard3` 升级到 `SuperClipboard` 时自动迁移旧数据
- 标题彩蛋：双击标题恢复「超级剪贴板」，三击变为「小楠の剪贴板」，五击变为「瑛楠の剪贴板」
- 一键检查更新（GitHub Releases）

## 技术栈

- **后端**：Rust、Tauri v2、SQLite（rusqlite）、arboard
- **前端**：React 19、TypeScript、Vite 8
- **存储**：SQLite，内容哈希去重

## 开发

```bash
# 安装依赖
pnpm install

# 开发模式运行
pnpm tauri:dev

# 生产构建
pnpm tauri:build
```

## 项目结构

```
src-tauri/
  src/
    clipboard.rs    # 剪贴板监控服务
    classifier.rs   # 内容类型分类
    storage.rs      # SQLite 存储层（条目 + 备忘录 + 设置）
    autostart.rs    # 开机自启动（Windows 注册表）
    window_position.rs # 设置入口默认位置计算（避开任务栏和屏幕边界）
    lib.rs          # Tauri 命令与应用初始化
    main.rs         # 入口文件
src/
  components/       # React UI 组件
    icons/              # 共享图标组件 (TrashIcon)
    SettingsButton.tsx  # 设置面板（语言、快捷键、开机自启、窗口置顶、备忘录）
    MemoList.tsx        # 备忘录列表（增删改查、自动保存、置顶、排序）
    MemoRichEditor.tsx  # 备忘录富文本/图片编辑器
    MemoBody.tsx        # 备忘录预览渲染
  api/              # Tauri 命令封装
    clipboard.ts    # 剪贴板 API
    settings.ts     # 设置 API
    memos.ts        # 备忘录 CRUD API
  i18n/             # 国际化（翻译文件 + Context）
  types/            # TypeScript 类型定义
```

## 后续规划

- [ ] **虚拟列表（Virtual Scrolling）**：当剪贴板条目积累较多时（数千条级别），当前 `.map()` 全量渲染方式会导致 DOM 节点过多、滚动卡顿。需引入虚拟列表（如 `@tanstack/react-virtual` 或 `react-window`），仅渲染可视区域内的条目，保持恒定渲染性能，配合无限滚动加载历史。
- [ ] **窗口跟随 / 记住位置**：保存窗口位置与插入符跟随功能已暂停开放，后续在交互体验稳定后再重新设计入口。
