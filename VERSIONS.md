# SuperClipboard 版本日志

## v2.0.0 (2026-07-01)

### 新功能

- **回收站**：备忘录软删除归档，归档 Tab 分「剪贴板」和「备忘录」子标签，30 天自动清理，显示剩余天数
- **正式命名**：软件名/项目名统一为 `SuperClipboard`，中文名为「超级剪切板」
- **标题彩蛋**：双击标题恢复，三击显示「小楠の剪切板」，五击显示「瑛楠の剪切板」
- **首次语言跟随系统**：首次启动自动按系统语言选择中英文，托盘菜单同步
- **备忘录图片粘贴**：编辑区支持 Ctrl+V 粘贴图片，预览区直接渲染
- **备忘录拖拽排序**：Pointer Events 实现，固定条目不参与拖拽
- **自定义备忘录颜色**：8 种预设色 + 自定义 HEX 输入，独立于主题色
- **剪贴板内容编辑**：编辑后显示最后编辑时间，原内容折叠保留
- **托盘重置位置**：右键托盘「设置」自动重置为默认位置
- **主题三段式切换**：浅色 / 深色 / 跟随系统

### 改进

- 「归档模式」更名为「回收站」，图标统一为废纸篓
- 分类算法升级为评分制，新增 Markdown、YAML/TOML 检测，减少误判
- 默认窗口位置改为右半屏居中，快捷键唤起保留上次位置
- 底栏按页面分别显示存储占用，高度统一
- 备忘录暗黑模式适配，标签颜色与剪贴板分类徽章一致
- 默认快捷键改为 Alt+X
- 从 `SuperClipboard3` 改名升级时自动迁移旧数据目录

### 修复

- 快捷键注册竞态、录制支持 Windows 键
- 单实例启动，修复重复点击快捷图标生成重复托盘图标
- 窗口位置保存状态与后端对齐，移动防抖
- 备忘录自定义颜色在樱花主题下不被覆盖
- 备忘录列表切换乱跳、hover 残留
- UTF-8 多字节字符截断崩溃

## v1.5.0 (2026-06-23)

### 新功能

- **备忘录功能**：独立模块，支持标题、正文、标签、搜索、置顶
- **剪贴板编辑**：编辑后显示编辑时间，原内容保留
- **原格式预览**：等宽字体显示原始内容
- **自动检查更新**、**窗口置顶**、**数据库大小显示**

### 改进

- 分类器重构为评分制，新增 CSS/HTML/SQL/Shell 检测
- 标签栏支持滚轮滑动

### 修复

- UTF-8 截断崩溃、备忘录切换不刷新

- 系统托盘常驻，自定义快捷键唤起

- 中英双语、设置面板、自动检查更新
